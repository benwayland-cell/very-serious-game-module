# Very Serious Game Jam Game (TODO update with final name)

A submission to The Very Serious Juniper Dev Game Jam.
It's a simple puzzle game where your controls get mixed up when you step on certain tiles.

## Instructions for Build and Use

Steps to build and/or run the software:

1. I have the build to my game saved online on itch.io at [https://bennyboy37.itch.io/the-dizzy-master](https://bennyboy37.itch.io/the-dizzy-master)
2. There you can either play it in the browser or download the .exe file
3. If you have the project file, you can also use Project -> Export

Instructions for using the software:

1. It's a simple puzzle game where you use WASD or arrow keys to move the player character
2. Your controls then get mixed up when you step on certain tiles.

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:
